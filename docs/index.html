Waktu Marketing Modal Config

Host this folder via GitHub Pages. The app reads /marketing-modal.json.

Live URL

marketing-modal.json

JSON Format

{
  "title": "Daily Quran Inspiration",
  "subtitle": "Add it from Lock Screen > Customize > Widgets > Waktu",
  "cta_text": "Got it",
  "slides": [
    {
      "title": "Daily Quran Widget",
      "subtitle": "One inspiring ayah every day.",
      "image_url": "https://your-domain/path/slide-1.jpg"
    },
    {
      "title": "Tap For Full Verse",
      "subtitle": "Open Waktu to see full details and translation.",
      "image_url": "https://your-domain/path/slide-2.jpg"
    }
  ]
}

Notes

  • Use image_url for remote images.
  • Use image_asset only for bundled app images.
  • At least one slide is required.
